Welcome to **World-of-Go**! 

This repository documents my journey of learning the Go programming language, containing code examples, notes, and my study roadmap.

## Repository Structure (仓库结构)
目前仓库包含以下部分：

- **`Stage1_BasicSyntax/`**: 
  - 包含 Go 语言所有的基础语法与用法的练习（变量、切片、Map、结构体等）。

- **`Stage2_Network_Web/`**: 
  - 接触 `net/http` 标准库，学习如何不依赖框架搭建 Web 服务器。
  - 学习 Context 上下文控制、JSON 数据的序列化与反序列化。

- **`Stage3_FrameworksAndEngineering/`**: 
  - 学习主流 Web 框架 **Gin** 和 ORM 框架 **GORM**。
  - 实践中间件编写、鉴权 (JWT) 以及 Redis 的使用。

- **`Stage4_DistributedAndMicroservices/`**: 
  - **(进阶)** 学习 gRPC 与 Protobuf 通信协议。
  - 接触 Docker 容器化部署与微服务基础概念。

- **`Projects/`**: 
  - 存放未来的实战项目源码（如短链接生成器、IM 聊天室）。

- **`Study_Plan.md`**: 
  - 学习路线文档。

- **`go.mod`**: 
  - Go 模块定义文件。
    
> 这是一个用于学习的仓库，大部分代码是手搓，格式和写法可能没那么规范，按个人习惯写的
