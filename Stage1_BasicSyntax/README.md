# Stage1_BasicSyntax：基础语法与核心特性

本目录是 Go 语言基础语法阶段的学习内容，包含**实验实战**、**章节源码**和**知识点笔记**。

## 📂 目录结构说明

### 1. `lab/` - 实验实战

- 每个子目录对应一个章节的实战练习（如 `lab1_基础语法`、`lab2_流程控制与函数`等）
- 包含 `labX_doc.md`（实验要求 + 步骤）和 `labX.go`（实验代码）
- 与 `src/` 中的章节一一对应，用于巩固知识点

### 2. `src/` - 章节源码

- 每个子目录对应一个核心章节（如 `chapter1_基础语法`、`chapter2_流程控制与函数`）
- 包含 `chapterX.go`（知识点代码示例）和 `chapterX.md`（章节知识点梳理）
- 与 `lab/` 中的实验一一对应，是实验的基础代码参考

### 3. `notes/` - 知识点笔记

- 按知识点拆分的 Markdown 文档（如编码、包机制、值类型等）
- 整理基础语法的核心概念、规则和细节，是理解 `src` 和完成 `lab` 的辅助资料
