### 📝 任务一：个人名片生成器 (IO与格式化)

**考察点**：`fmt.Printf` vs `fmt.Sprintf`，占位符的使用

请编写代码，模拟生成一张标准的“控制台名片”。

1. 定义以下变量（使用不同的声明方式：`var` 和 `:=`）：
   - 姓名 (String): 自定义
   - 年龄 (Int): 自定义
   - 是否在职 (Bool): 自定义
   - 期望薪资 (Float64): 例如 25000.50
2. **要求 A**：使用 `fmt.Printf` 在控制台打印一行表头：`Name | Age | IsActive | Salary` (中间使用 `\t` 制表符对齐)。
3. **要求 B**：使用 `fmt.Sprintf` 将上述变量格式化为一个字符串 `infoStr`。
   - 要求：姓名两边加双引号（`%q`），薪资保留两位小数（`%.2f`）。
4. **要求 C**：最后打印 `infoStr`。

### 🛒 任务二：超市收银系统 (类型转换与默认值)

**考察点**：零值机制、强类型计算、类型转换

Go 语言禁止不同类型的变量隐式计算（例如 `int` * `float`），请编写代码完成以下计算：

1. 声明变量 `price` (单价) 为 `float64`，赋值 `19.9`。
2. 声明变量 `count` (数量) 为 `int`，赋值 `5`。
3. 声明变量 `total` (总价) 为 `float64`，**不赋值**（利用零值机制）。
4. **计算逻辑**：尝试计算 `total = price * count`。
   - *提示：直接乘会报错，你需要修正代码使得编译通过。*
5. **输出结果**：打印“购买了 [count] 件商品，单价 [price]，总共花费 [total]”。
6. **思考题（写在注释里）**：如果声明一个 `string` 类型的变量但没赋值，直接打印它，控制台会显示什么？

### 🚥 任务三：权限管理模拟 (位运算与逻辑符)

**考察点**：位运算符 (`<<`, `|`, `&`, `^`) 的实际应用

在系统开发中，我们常用二进制位来表示权限。

1. 定义三个常量（使用 `const` + `iota` + 左移运算 `<<`）：
   - `Readable`   (代表二进制 001)
   - `Writable`   (代表二进制 010)
   - `Executable` (代表二进制 100)
2. 定义一个变量 `userPerm` (用户权限)，赋予它 **“可读 且 可写”** 的权限（提示：使用位或 `|`）。
3. **检查权限**：
   - 使用位与 `&` 运算检查 `userPerm` 是否拥有 `Executable` 权限。
   - 如果拥有，打印 "Permission Allowed"；否则打印 "Permission Denied"。
4. **添加权限**：
   - 使用位运算给 `userPerm` 增加 `Executable` 权限。
   - 再次打印 `userPerm` 的整数值（应该是 7）。

### 💾 任务四：硬盘单位转换器 (iota 表达式继承)

**考察点**：`iota` 的表达式继承特性

请参考笔记中 `1 << iota` 的逻辑，编写一个能够打印存储单位字节数的程序。

1. 定义一个常量组：
   - 第一行定义 `_` (忽略 0，因为我们想从 KB 开始算，或者你可以定义 B=1)。
   - 使用 `iota` 和左移运算定义 `KB`, `MB`, `GB`, `TB`。
   - 规则：`KB` 应该是 1024 (2^10)，`MB` 是 1024*1024 (2^20)...以此类推。
2. **要求**：**只能在第一行写表达式**，后面的 `MB, GB, TB` 必须利用 Go 的**隐式继承**机制自动生成。
3. 输出：打印 `1GB` 对应的字节数是多少。
